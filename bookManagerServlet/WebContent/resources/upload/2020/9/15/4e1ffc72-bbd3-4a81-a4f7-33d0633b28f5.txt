
1) 역할

 AWS, Oracle Cloud 세팅 및 개발 총괄 : 김훈
 회원가입/로그인, 마이페이지, 관리자페이지 구현 : 문지영, 윤세민
 운동 구현 : 장성현
 운동 및 이벤트 구현 : 장연호
 쇼핑 및 이벤트 구현 : 박주이 이인지

 ===== ===== =====

2) 개요

 구글의 오픈소스 머신러닝인 teachable machine 2.0을 이용하여
 사용자의 운동수행에 포인트를 부여하는 동시에 관련 제품을 판매한다.

 ===== ===== =====

3) 구현기능

 머신러닝이 사전에 학습한 운동별 모션과 대조하여 사용자의 운동수행을 평가
 운동 및 이벤트(출석, 신기록 경쟁 등) 수행에 포인트 부여
 헬스케어 관련 제품 판매, 후기등록, 반품 기능
 정보 공유를 위한 게시판, 파일첨부, 댓글, 추천, 신고 기능

 ===== ===== =====

4) 설계시 주안점

 건강을 챙기는 동시에 실리적인 이득도 볼 수 있게 하여 지속적인 접속 유도

 ===== ===== =====

5) 사용 기술 및 개발

 OS : Window10
 IDE : 구름IDE
 DB : Oracle Cloud
 Server : AWS(Redhat)
 WAS : Apache Tomcat
 사용 기술 :  java, jsp, javaScript, jquery, ajax, bootstrap, spring, mybatis, html, css, tailwind, alphineJS, teachable machine 2.0
 협업 : Git, Trello, Slack
